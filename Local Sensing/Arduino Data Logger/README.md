# What is in this folder?
This folder basically contains the arduino and python interface files I used for this project. To run them you will have to open
the `.cpp` files in **platformio**, as that is where I created them. As for the python interface, just the python enviorment in vscode is fine. Make sure
you have libraries such as PyQt and stuff installed. I havent really taken the time to clean this up, as this was just a side project to the main theme
of landslide prediction. I would reccomend you instead look into the Remote Sensing methods, as they have more potential, and can be expanded greatly. 
But then again, feel free to look at what you want to.
